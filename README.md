## 📌 Overview

A demo project showcasing a complete CI/CD pipeline — every push to main automatically builds a Docker image and deploys the Node.js app via GitHub Actions.

---

## 🔁 Pipeline Flow

Push to main → GitHub Actions triggers → Docker builds image → App deployed

---

## 🚀 Getting Started

```bash
git clone https://github.com/SnehashreeHazra/cicd-demo.git
cd cicd-demo
npm install
node index.js
```
